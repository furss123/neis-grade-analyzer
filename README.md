# 나이스 성적 분석기

나이스에서 내려받은 서로 다른 형태의 성적 Excel 파일을 브라우저에서만 읽어 공통 데이터 구조로 변환하고 분석하는 React 웹 앱입니다.

## 주요 기능

- `.xlsx`, `.xls` 다중 파일·폴더 업로드
- 셀 내용 특징 점수 기반 문서 유형 판별
- 병합 셀, 가변 헤더, 과목명·학급/번호 정규화
- 정기시험·수행평가·학기말 데이터 표준화와 출처 우선순위 병합
- 확보된 필드에 따른 분석 메뉴 자동 활성화
- 담임·교과교사 대시보드, 시험 변화, 성취도, 학생 프로필
- 학생 개별 심층 성적 분석과 개별·전체 학생 인쇄 리포트
- 일반/익명 Excel 내보내기와 브라우저 PDF 인쇄
- 낮은 신뢰도 파일의 유형·학년도·학기·학년·반 보정

파일은 서버로 전송하지 않으며 브라우저 메모리에서만 처리됩니다. 실제 학생 파일을 저장소에 커밋하지 마세요.

## 실행

```bash
npm install
npm run dev
```

검증은 `npm test`, 프로덕션 빌드는 `npm run build`로 실행합니다.

## GitHub Pages

저장소 설정의 **Pages → Source**를 **GitHub Actions**로 선택합니다. `main` 브랜치에 푸시하면 테스트와 빌드 후 `.github/workflows/deploy.yml`이 배포합니다. Vite의 상대 경로 base 설정으로 사용자/프로젝트 Pages 양쪽에서 동작합니다.

## 지원 문서 유형

1. 정기시험 학급별 일람표
2. 정기시험 교과목별 일람표
3. 전과목 성적 일람표 (지필·수행평가 및 학기말 점수 포함)
4. 과목 정기시험·수행평가 성적 일람표
5. 학기말 성적 종합일람표 (기존 형식 호환)

학교별 Excel 출력 구조가 다를 수 있습니다. 자동 판별 신뢰도가 낮을 때 업로드 결과에서 문서 유형과 기본 정보를 보정할 수 있습니다.
